
## Setup

- Clone the repo
- `$ git checkout notes-api` **note!! please checkout into notes-api branch, otherwise it will not work**
- Run `$ npm install`
- Then `$ node server.js`

## Test
- With mocha-cli installed (`$ npm i mocha -g`)
- Start server : `node server.js`
- Run : `mocha test/main.spec.js`
